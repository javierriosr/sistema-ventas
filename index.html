💼 Sistema de Ventas e Inventarios

Este sistema está diseñado para gestionar de forma eficiente tus ventas, productos e inventario. Ideal para negocios que buscan una herramienta simple, funcional y lista para usar desde Windows.

  • ✅ Registro de productos con stock
  • ✅ Control de ventas con historial
  • ✅ Base de datos local (SQLite)
  • ✅ Interfaz amigable y portable
